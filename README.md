\# 💊 Mining the Matrix: AI for Drug Safety



\## Project Overview

An AI-powered drug safety monitoring system that predicts potential drug-drug interactions and provides safety recommendations.



\## Features

\- 🔍 Search any medicine for safety information

\- 🔮 Predict drug interactions with risk levels

\- 📊 Interactive analytics dashboard

\- 💡 Safety recommendations
